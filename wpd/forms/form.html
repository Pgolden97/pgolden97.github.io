THIS IS THE EXPLANATION OF THE FORM:


In this example, we are using SVG Image files and CSS to sylize the form, As evident with the check boxes, the drop down menu icon, the email icon in the email feild being SVG files. Everything is stylized in the css document that is linked to the html. For example, in order to affect the email feild we use the selector of "form#signup input#email" being that we are working in the signup form, in an input feild, and the name of the feild is email. Anything under that selector will affect only that location under the form. In order to affect a state of one of the check boxes similar to the hover, we would use something like "form#signup input[type="checkbox"] + label" which will in return target the form signup, an input (ex. feild or checkbox) and then in the [] we put the type of input that we are targeting and then we can add another target by using the plus so in this case we also add the label as a target to be affected by the css.
